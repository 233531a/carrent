# CarRent (Rentify) — система аренды автомобилей
CarRent — учебный, но достаточно реалистичный веб‑сервис для компании по аренде автомобилей.
Через браузер можно посмотреть каталог машин, подобрать авто под свои нужды, оформить бронирование и управлять автопарком на уровне администратора.​

### Что умеет система:

1. Показывать каталог автомобилей с фильтрами по марке, классу, коробке передач и диапазону цен.​
2. Разделять машины по типу использования: обычная аренда, такси, доставка.​
3. Отображать подробную карточку автомобиля с характеристиками и ставкой в сутки.
4. Оформлять бронь: выбор дат, проверка пересечений по времени, автоматический расчёт итоговой стоимости.​
5. Давать клиенту личный кабинет с историей заявок и возможностью отмены активных бронирований.
6. Предоставлять админ‑панель для управления пользователями, ролями и автопарком (создание, редактирование, удаление машин и броней).​
7. Ограничивать доступ в зависимости от роли (CLIENT, EMPLOYEE, MANAGER, ADMIN) на базе Spring Security.​

### Технологический стек

* Java 17, Spring Boot (Web, MVC, Security, Data JPA, Validation).​
* Hibernate / JPA для ORM и работы с MySQL.​
* Thymeleaf для серверного рендеринга HTML‑страниц.
* Maven для сборки и управления зависимостями.
* HTML, CSS, JavaScript для клиентской части.

## Архитектура проекта

**Проект построен по классической многослойной схеме:**

* model — JPA‑сущности: пользователи, роли, клиенты, автомобили, аренды, статусы.
* repository — интерфейсы Spring Data JPA для доступа к базе.
* service — бизнес‑логика: проверка доступности авто, расчёт стоимости аренды, управление пользователями и ролями.
* controller — веб‑и REST‑контроллеры (Thymeleaf‑страницы и JSON‑API для работы с автомобилями).
* config — настройка безопасности и подключения к БД.
* templates / static — HTML‑шаблоны интерфейса и статические ресурсы (CSS, JS, изображения).
* Отдельный CarRestController предоставляет JSON‑эндпоинты для каталога машин — их можно использовать для мобильного клиента или внешних интеграций.
